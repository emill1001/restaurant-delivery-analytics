# Business Insights

## Customer Behaviour

- Customers placed multiple orders throughout the observed period.
- Some customers requested modifications such as exclusions and extras, indicating demand for customized orders.

---

## Product Performance

- Certain pizza types were ordered significantly more often than others.
- Popular menu items generated the majority of total sales volume.

---

## Delivery Performance

- Most deliveries were completed successfully.
- Delivery distance and duration required data cleaning before reliable analysis.

---

## Data Quality

Several inconsistencies were identified in the original dataset:

- Missing values
- Text stored instead of numeric values
- Units mixed with numbers (km, minutes)
- Empty strings representing NULL values

These issues were corrected before analysis to improve result accuracy.

---

## Conclusion

This project demonstrates an end-to-end SQL analysis workflow including data cleaning, data transformation, and business analysis to support decision-making.
