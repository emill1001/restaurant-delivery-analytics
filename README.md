# Restaurant Delivery Analytics

## Project Overview

This project analyzes the operational performance of a restaurant delivery business using SQL. The analysis includes data cleaning, transformation, and business-oriented queries to extract insights about customer behavior, delivery performance, and product demand.

---

## Objectives

- Clean and prepare raw data for analysis
- Analyze customer ordering patterns
- Evaluate delivery performance
- Identify best-selling products
- Generate business insights using SQL

---

## Tools

- SQL
- Relational Databases

---

## Skills Demonstrated

- Data Cleaning
- Data Transformation
- Data Validation
- SQL Joins
- Aggregate Functions
- CASE Statements
- Common Table Expressions (CTEs)
- Business Analysis

---

## Business Questions Answered

- How many pizzas were ordered?
- Which pizzas were the most popular?
- Which customers placed the most orders?
- How many successful deliveries were completed?
- Which runner completed the most deliveries?
- What was the average delivery time?
- How did order changes affect deliveries?

---

## Key Learning Outcomes

This project strengthened my SQL skills by applying data cleaning, transformation, and analytical techniques to solve real business problems using a relational database.
